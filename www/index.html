Introducing dRunner

Making Docker easier.

Features Install

Features You'll Love.

dRunner was designed to be a simple and intuitive way to install, configure and start containerised services from the command line.

01.

Cross-platform

dRunner is available natively for Windows, Linux (Debian and Ubuntu) and Mac OS X.

02.

Self-contained

dRunner compatible Docker Images (dServices) are self contained - everything dRunner needs is inside.

03.

Backup/Restore

Backup an entire dService to a single encrypted file, trivially restore on another machine - whatever the OS.

04.

Discoverable commands

Installed dServices are directly available on the command line with help right there. Basic commands and configuration are standardised across dServices and very simple.

05.

Easy To Customize

Turning an existing Docker container into a dService often only requires a single lua file. The minecraft dService gives a real example.

06.

Simple

dRunner is a single executable requiring only Docker. Destroying a service leaves the machine in a clean state. It has a small footprint: just a single folder plus service Docker Volumes.
